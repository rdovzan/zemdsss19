## Setup

- Install Python3
- Install [Kaggle API](https://github.com/Kaggle/kaggle-api) --> pip3 install kaggle --upgrade
- Go to kaggle.com --> Profile --> Create API token
- Move 'kaggle.json' to ~/.kaggle/
- Run 'kaggle -v'
